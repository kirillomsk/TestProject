
Устанавливаем субд postgresql (https://www.postgresql.org/download/)
В pgAdmin создаём сервер с параметрами
database="postgres", host="127.0.0.1", port='5432', user="postgres", password="123456789

/chromedriver поддерживает версию Google Chrome 88.0.4324.182/

Запускаем файл One.py в котором находится первое задание
    собираем данные из таблицы и записываем в бд
    заполняем таблицу рандомными данными
    снова собираем данные из таблицы на сайте и сравнием с данными из бд
    в консоль выводим пункты, которыми таблицы отличаются

Автотесты запускаются командой 'pytest' в терминале (если проект открыт в pycharm)
в противном случае в командной строке необходимо активировать окружение
    "Путь:\до\папки с проектом\TestProject\venv\Scripts\activate.bat"
    и из папки TestProject запустить pytest
Тесты:
    test_one.py - ui тесты для первого задания
    test_aip.py - тесты для второго задания

в conftest.py хранится:
    url для api
    путь до chromedriver

некоторые тесты упадут т.к. у таблицы на сайте есть баги
